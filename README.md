# Movie Recommendation System

A content-based movie recommendation system that suggests similar movies based on user selection.  
The system uses **text vectorization and cosine similarity** to find related movies.


## Features
-  Recommends movies similar to the selected movie.
-  Fast similarity computation.
-  Simple and interactive UI (Streamlit).

## Tech Used
-  **Python**
-  **Pandas, Numpy**
-  **Scikit-learn**
-  **Cosin Similarity**
